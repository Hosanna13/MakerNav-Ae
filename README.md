Navigating markers is time consuming. There's probably a plugin out there that does this. That plugin probably costs twenty dollars — and that's twenty dollars I am not willing to spend.

Introducing... _drumroll please_ ... **MarkerNav**.

A free, lightweight panel for jumping between composition markers so you don't have to scrub through the timeline once you run out of keyboard shortcuts past marker 9.
</div>

---

# 📖 Overview 

<div align="left" style="max-width: 600px;">

MarkerNav is not a complicated tool for Adobe After Effects. It is a simple, focused panel that fits right into your editing workflow. It lets you jump between every comp marker in one click, looping infinitely in both directions so you never hit a dead end.

</div>


# 🫣 Key Features: 

<div align="left" style="max-width: 600px;">

- `Navigate` **forward** and **backward** through all comp markers with two buttons
- `Infinite looping wraps` from the **last** marker **back** to the **first** (and vice versa)
- `Always shows` **"Marker X / Y"** so you know exactly where you are
- `Syncs to your playhead`, scrub anywhere and Prev/Next picks up from there
- `Works` with **any markers** you already have, no setup needed

</div>

# 😛 How to Use

<div align="left" style="max-width: 600px;">

1. **Drop**: Add markers to your comp (`Numpad * `at any time)
2. **Navigate**: Click `◀ Prev `or `Next ▶ `to jump between them
3. **Cut**: Split your layers however you like at each stop

</div>

# 🧑🏾‍💻 Installation: 

## 📂 Method 1: Manual Copy

Copy the `MarkerNav.jsx` file into the appropriate directory for your operating system:
</div>


<table>
    <tr>
        <th> MAC</th>
    </tr>
    <tr>
        <td>/Applications/Adobe After Effects &lt;ver&gt;/Scripts/ScriptUI Panels/ </td>
    </tr>
    <tr>
        <th> Windows </th>
    </tr>
    <tr>
        <td> C:\Program Files\Adobe\Adobe After Effects &lt;ver&gt;\Support Files\Scripts\ScriptUI Panels\</td>
    </tr>
</table>



<div align="left" style="max-width: 600px;">

## 🗂️ Method 2: Inside After Effects

Install it directly inside the application:

1. Go to **File** > **Scripts** > **Install ScriptUI Panel...**
2. Select `MakerNav.jsx`
3. Restart After Effects.
4. Open the panel by navigating to **Window** > **MakerNav**.
